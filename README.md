 # 🧮 Python Calculator GUI

A simple and user-friendly calculator application built using Python and Tkinter. This project performs basic arithmetic operations with a clean graphical interface.


## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧼 Clear button (C)
- 📱 Simple and responsive GUI
- 🎨 Attractive button design (light blue theme)


## Technologies Used

- Python
- Tkinter (GUI Library)
